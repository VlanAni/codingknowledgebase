`http.ServeMux` - это **структура**, которая каждому URL сопоставляет определённый паттерн, и для наиболее подходящего паттерна вызывается соответствующий обработчик, который необходимо зарегистрировать 

> [!info] Docs 
> ServeMux is an HTTP request multiplexer. It matches the URL of each incoming request against a list of registered patterns and calls the handler for the pattern that most closely matches the URL

Регистрируются обработчики с помощью методов `Handle` и `HandleFunc`: 
``` go
func (mux *ServeMux) Handle(patternstring, handler Handler))
```

``` go
func HandleFunc(pattern string, handler func(ResponseWriter, *Request)) 
```

По умолчанию существует `DefaultServeMux`, обработчики и паттерны для которого назначаются с помощью функций `HandleFunc` из пакета `net/http`
