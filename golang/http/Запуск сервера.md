Для запуска сервера применяется функция `ListenAndServe`:

``` go
func ListenAndServer(addr string, handler Handler) error
```

> [!info] Docs
> ListenAndServe listens on the TCP network address addr and then calls [Serve](https://pkg.go.dev/net/http#Serve) with handler to handle requests on incoming connections. Accepted connections are configured to enable TCP keep-alives.
> 
> The handler is typically nil, in which case [DefaultServeMux](https://pkg.go.dev/net/http#DefaultServeMux) is used.
> 
> ListenAndServe always returns a non-nil error.
