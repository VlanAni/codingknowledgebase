В стандартной библиотеке `Go` существует **два способа** обрабатывать `http`-запросы к серверу: интерфейс `Handler` и функция `HandleFunc` 
### Интерфейс `http.Handler`

Интерфейс `http.Handler` объявляет метод `ServeHTTP`, который *должен* отправлять ответ на запрос: 
``` go
type Handler interface {
	ServeHTTP(http.ResponseWriter, r *http.Request)
}
```

> [!info] Official documentation
> Handler.ServeHTTP should write reply headers and data to the [ResponseWriter](https://pkg.go.dev/net/http#ResponseWriter) and then return. Returning signals that the request is finished; it is not valid to use the [ResponseWriter](https://pkg.go.dev/net/http#ResponseWriter) or read from the [Request.Body] after or concurrently with the completion of the ServeHTTP call.
> 
> Depending on the HTTP client software, HTTP protocol version, and any intermediaries between the client and the Go server, it may not be possible to read from the [Request.Body] after writing to the [ResponseWriter](https://pkg.go.dev/net/http#ResponseWriter). Cautious handlers should read the [Request.Body] first, and then reply.
> 
> Except for reading the body, handlers should not modify the provided Request.
> 
> If ServeHTTP panics, the server (the caller of ServeHTTP) assumes that the effect of the panic was isolated to the active request. It recovers the panic, logs a stack trace to the server error log, and either closes the network connection or sends an HTTP/2 RST_STREAM, depending on the HTTP protocol. To abort a handler so the client sees an interrupted response but the server doesn't log an error, panic with the value [ErrAbortHandler](https://pkg.go.dev/net/http#ErrAbortHandler).
### `http.HandleFunc`

`HandleFunc` - это **функция**, которая принимает на вход **URL**, для которого нужно зарегистрировать обработчик, а также саму **функцию-обработчик**

Сигнатура: 
``` go
func HandleFunc(pattern string, handler func(ResponseWriter, *Request))
```

> [!info] Official documentation
> HandleFunc registers the handler function for the given pattern in [DefaultServeMux](https://pkg.go.dev/net/http#DefaultServeMux). The documentation for [ServeMux](https://pkg.go.dev/net/http#ServeMux) explains how patterns are matched.
