### Function

``` java
R apply(T arg); 
```

Имеет один метод, который принимает на вход аргумент типа `T` и возвращает значение типа `R` 
### Consumer 

``` java
void accept(T arg); 
```

Принимает аргумент типа `T` и ничего не возвращает 
### Supplier 

``` java
T get(); 
```

Поставляет значения типа `T`
### Predicate 

``` java
boolean test(T arg); 
```

Принимает на вход значение типа `T` и проверяет выполнимость какого-либо условия на этом значении
