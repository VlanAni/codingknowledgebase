## Checked exceptions

Это исключения, обработка которых проверяется на этапе компиляции

> [!info] Исключения в методе 
> Если метод может выбросить проверяемое исключение, то компилятор проверяет, обрабатывает ли метод это исключение в блоке `try-catch` / использует ключевое слово `throw` с этим исключением в своей сигнатуре

## Unchecked exceptions 

Выражают критические проблемы в программе и не обязаны обрабатываться программистом 

Компилятор не проверяет обработку этих исключений
