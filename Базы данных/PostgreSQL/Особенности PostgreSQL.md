**PostgreSQL** - база данных с **клиент-серверной архитектурой** 

Отправка **запроса** к серверу базы данных и получение **ответа** от базы данных происходит по протоколу поверх стека **TCP/IP** 

Для работы с БД PostgreSQL необходима клиентская программа и специальная библиотека для языка программирования
