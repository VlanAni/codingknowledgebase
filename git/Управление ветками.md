Для управления ветками используется команда `git branch`

Это команда позволяет: 
- удалять ветки (с помощью флага `-D`) 
- просматривать ветки (с помощью флага `--list`) 
- создавать ветки 
### Слияние веток 

Для слияния веток используется команда `git merge [branch_to_merge]`  
##### Стратегии мёржа 

- `Fast forward` - передвигаем указатель ветки 
- `recursive` - это способ слияния, при котором мы не можем передвинуть голову ветки, а вместо этого нужно найти общий коммит, посчитать результирующее изменение и сделать новый коммит, который называется **коммитом слияния**
