Для управления ветками используется команда `git branch`

Это команда позволяет: 
- удалять ветки (с помощью флага `-D`) 
- просматривать ветки (с помощью флага `--list`) 
- создавать ветки 
### Слияние веток 

Для слияния веток используется команда `git merge [branch_to_merge]`
