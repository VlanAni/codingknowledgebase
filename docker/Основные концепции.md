## Легковестность

`docker`-контейнеры намного легче и менее требовательные, чем виртуальные машины 
## Образ контейнера `docker`

**Образ** - это шаблон, по которому надо сбирать контейнер 

Образ содержит: 
- образ базовой ОС 
- код приложения 
- необходимые зависимости 
## `Dockerfile`

`Dockerfile` - текстовый файл, который содержит команды, которые позволяют собрать контейнер из образа 

Этот файл содержит описание базового образа (исходный слой образа), на который можно накладывать новые слои 
## Контейнер `Docker` 

**Контейнер** - это то, что собирается из образа и запускается
